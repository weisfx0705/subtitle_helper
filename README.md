# 翻譯寫手的書僮 (Subtitle Helper)

**一款為專業翻譯與寫作者設計的智慧工具**

這是一個專為處理帶有文學細微差別的字幕檔案（.srt）而設計的網頁應用程式。它結合了 Google强大的 Gemini AI 模型，利用您提供的故事背景、角色設定與語言風格，生成自然、高品質且完全尊重原始時間軸的翻譯成果。

## ✨ 功能特色

- **📂 支援 .srt 字幕檔**：直接上傳字幕檔案，即刻開始作業。
- **🧠 智慧情境翻譯**：不只是逐字翻譯，更參考「故事大綱」、「角色介紹」與「語言風格」進行深度翻譯。
- **📝 雙欄對照介面**：左側原文、右側譯文，方便即時校對與修改。
- **🔄 靈活的重譯功能**：
  - 針對特定段落重新翻譯。
  - 修改完譯文後，可將「已修改的譯文」作為新的原文，進行第二輪精修（例如：先翻成初稿，再潤飾成口語）。
- **💾 匯出功能**：一鍵下載保留原始時間碼的翻譯版 .srt 檔案。
- **🔒 隱私保障**：您的 API Key 僅儲存於本地瀏覽器端，不會上傳至任何第三方伺服器。

## 🚀 快速開始

### 1. 前置準備
您需要一組 Google Gemini API Key。
- 前往 [Google AI Studio](https://aistudio.google.com/) 申請 API Key。

### 2. 安裝與執行
本專案使用 React + Vite 開發。若您想在本地端執行：

```bash
# 安裝相依套件
npm install

# 啟動開發伺服器
npm run dev
```

開啟瀏覽器並訪問 `http://localhost:5173` 即可使用。

### 3. 建置專案
若要打包成靜態檔案（支援單一 HTML 輸出）：

```bash
npm run build
```

## 📖 使用教學

1. **輸入金鑰**：首次進入請點擊右上角設定，輸入您的 Gemini API Key。
2. **上傳檔案**：點擊首頁或拖曳上傳您的 `.srt` 字幕檔。
3. **設定情境**：
   - **故事大綱**：簡述影片內容或劇情走向。
   - **角色介紹**：列出主要角色的個性、說話習慣。
   - **語言設定**：選擇來源語言與目標語言（甚至可以指定如「台灣口語」、「古文風格」等）。
   - **模型選擇**：選擇即將使用的 Gemini 模型版本。
4. **開始翻譯**：系統將分批次處理字幕，請耐心稍候。
5. **校對與匯出**：翻譯完成後，您可以在介面上直接修改內容，最後點擊「下載字幕」保存成果。
