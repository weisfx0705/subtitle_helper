# AI Subtitle Helper (字幕翻譯助手)

這是一個基於 Google Gemini AI 的字幕翻譯工具，專為追求高品質翻譯的字幕組與創作者設計。它不只是將文字從 A 語言轉為 B 語言，而是透過理解「劇情脈絡」與「角色設定」，提供如同專業譯者般的精準翻譯。

## 🌟 核心特色：情境感知翻譯 (Context-Aware Translation)

本專案最大的特點在於深度整合了 **Google Gemini 3 Flash Preview** 的理解能力，解決了傳統機器翻譯的痛點：

### 1. 劇情與角色連貫性
在翻譯開始前，您需要輸入「劇情大綱」與「角色介紹」。AI 會讀取這些資訊，確保：
- **角色語氣精準**：區分長幼尊卑、正式或口語（例如：對長輩的敬語、朋友間的俚語）。
- **專有名詞統一**：根據劇情背景，正確翻譯特定術語或名稱。
- **情感捕捉**：理解對話背後的潛台詞，而非僅翻譯字面意思。

### 2. 時間軸適配 (Time-Constraint Awareness)
翻譯字幕最難的挑戰之一是「字數限制」。
- 本工具會計算每句字幕的**顯示時間 (Duration Seconds)**。
- AI 會在翻譯時自動調整句式長短，確保觀眾在有限時間內能舒適地閱讀完畢，避免翻譯過長造成的閱讀負擔。

### 3. 文化與語境轉譯
- 拒絕生硬的直譯 (Literal Translation)。
- 強調**信、達、雅**，將源語言的文化梗或隱喻，轉換為目標語言受眾能理解的表達方式。
